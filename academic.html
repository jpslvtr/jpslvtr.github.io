Academic


  • Naval Vessel Classification (CS 231n): Compared performance of Google's Vision Transformer (ViT) and Microsoft's Swin Transformer models for maritime vessel classification. Achieved 92.54% accuracy through fine-tuning with an optimal learning rate of 5e-5. Analyzed 8,908 ship images across cargo, military, carrier, cruise, and tanker categories.
  • Ship Arrival Regression Analysis (OIT 272): Developed predictive models for Saqr Port in Ras Al Khaimah using AIS data to improve arrival time predictions. Implemented linear regression, regression trees, and random forest models to analyze arrival patterns across 904 unique vessels. The project aimed to reduce costly demurrage fees through better resource allocation.
  • Web Security (CS 155): Created and defended against seven web attack vectors including cookie theft, cross-sit request forgery, SQL injection, and profile worms in a Node.js/SQLite application. Implemented security measures including CSRF tokens, cookie signing, and input sanitization.
  • Network Security (CS 155): Implemented port scanning detection and ARP spoofing detection systems using Go. Created a DNS spoofing tool that hijacks HTTP connections through a three-stage attack: ARP spoofing, DNS response manipulation, and HTTP request interception. Developed using gopacket for packet manipulation and analysis.
  • Quantum Mechanics & Free Will (APPPHYS 363): Explored how quantum mechanics impacts determinism. Examined how quantum indeterminacy affects prediction models compared to classical physics, using Conway's Game of Life to demonstrate emergent properties and analyzing implications for human agency.
