Academic


  • CS231N Deep Learning for Computer Vision: Ship Classification Using Vision Transformers. Compared performance of Google's Vision Transformer (ViT) and Microsoft's Swin Transformer models for maritime vessel classification, achieving 92.54% accuracy through fine-tuning with an optimal learning rate of 5e-5. Analyzed 8,908 ship images across cargo, military, carrier, cruise, and tanker categories.
  • OIT 276 Data and Decisions: Ship Arrival Prediction System. Developed predictive models for Saqr Port in Ras Al Khaimah using AIS data to improve arrival time predictions. Implemented linear regression, regression trees, and random forest models to analyze arrival patterns across 904 unique vessels. The project aimed to reduce $20,000/hour demurrage fees through better resource allocation.
  • CS155 Computer and Network Security - Web Security Analysis: Created and defended against seven web attack vectors including cookie theft, cross-site request forgery, SQL injection, and profile worms in a Node.js/SQLite application. Implemented comprehensive security measures including CSRF tokens, cookie signing, and input sanitization.
  • CS155 Computer and Network Security - Network Security: Implemented port scanning detection and ARP spoofing detection systems using Go. Created a DNS spoofing tool that hijacks HTTP connections through a three-stage attack: ARP spoofing, DNS response manipulation, and HTTP request interception. Developed using gopacket for packet manipulation and analysis.
  • APPPHYS 363: Modern Physics & Literature: Quantum Mechanics & Free Will. Explored how quantum mechanics impacts determinism through analysis of emergence in thermodynamics, computing, and biological systems. Examined how quantum indeterminacy affects prediction models compared to classical physics, using Conway's Game of Life to demonstrate emergent properties and analyzing implications for human agency.
